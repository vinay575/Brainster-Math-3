📋 PREREQUISITES
  ✓ Node.js v16+
  ✓ TiDB Cloud account
  ✓ AWS S3 bucket
  ✓ Firebase project (for Google login)

🚀 SETUP (5 MINUTES)

1. INSTALL BACKEND
   cd backend
   npm install

2. CONFIGURE ENVIRONMENT
   cp .env.example .env
   # Edit .env with your credentials

3. DOWNLOAD TiDB CERTIFICATE
   # Place ca.pem in project root

4. INITIALIZE DATABASE
   npm run seed

5. START BACKEND
   npm start
   # Server: http://localhost:5000

6. START FRONTEND
   cd ../frontend
   npx serve .
   # Frontend: http://localhost:3000

7. UPLOAD VIDEOS TO S3
   # Naming: L{level}_{start}_{end}.mp4
   # Example: L1_1_5.mp4

8. SYNC VIDEOS
   # Login as admin → Videos → Sync S3
